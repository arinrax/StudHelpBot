# StudHelpBot: Ассистент для выбора тем ВКР

## Описание
Телеграм-бот для помощи студентам в подборе тем курсовых и выпускных квалификационных работ. Система использует локальную языковую модель для диалога, семантический поиск для понимания смысла запросов и машинное обучение для прогнозирования оценки

## Основные возможности
- **Семантический поиск**: Поиск тем по смыслу с помощью векторных представлений и кэшированием индексов
- **Диалог**: Обработка запросов через локальную LLM с механизмом Function Calling
- **Анализ руководителей**: Получение статистики, списка работ и средней оценки по научному руководителю
- **Генерация идей**: Подбор случайных тем и поиск похожих формулировок
- **Прогнозирование оценки**: Небольшая учебная ML-модель для предсказания вероятной оценки на основе темы, руководителя и среднего балла
- **Логирование**: Структурированная запись событий для мониторинга и отладки

## Технологический стек
- **Язык**: Python 3.9+
- **Telegram API**: pyTelegramBotAPI
- **LLM**: Ollama (Qwen 2.5 7B) через OpenAI API
- **ML**: Sentence-Transformers (SBERT), NumPy, scikit-learn
- **База данных**: SQLite3
- **Переменные окружения**: python-dotenv
